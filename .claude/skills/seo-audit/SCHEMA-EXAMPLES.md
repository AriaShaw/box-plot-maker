# Schema Markup Examples

Ready-to-use Schema.org structured data templates for tool landing pages.

## SoftwareApplication Schema (Required)

Use for all tool pages. Tells Google this is a software application.

```json
{
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  "name": "Viking Name Generator",
  "description": "Generate authentic Viking names based on historical Norse naming traditions. Free online tool for creating warrior names, epithets, and Old Norse transliterations.",
  "applicationCategory": "UtilitiesApplication",
  "operatingSystem": "Web browser",
  "offers": {
    "@type": "Offer",
    "price": "0",
    "priceCurrency": "USD"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "ratingCount": "127"
  }
}
```

### Next.js Implementation

```typescript
// app/page.tsx or app/layout.tsx
export default function Page() {
  const schema = {
    "@context": "https://schema.org",
    "@type": "SoftwareApplication",
    "name": "Your Tool Name",
    "description": "Your tool description",
    "applicationCategory": "UtilitiesApplication",
    "operatingSystem": "Web browser",
    "offers": {
      "@type": "Offer",
      "price": "0",
      "priceCurrency": "USD"
    }
  };

  return (
    <>
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(schema) }}
      />
      {/* Your page content */}
    </>
  );
}
```

## FAQPage Schema (Recommended)

Use for FAQ sections. Enables rich snippets in Google search.

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "What is a Viking name generator?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "A Viking name generator is a free online tool that creates authentic Norse warrior names based on historical Viking Age naming conventions. It combines real Old Norse elements with historically documented epithets to produce names that Vikings actually used during the 8th to 11th centuries."
      }
    },
    {
      "@type": "Question",
      "name": "How accurate is this Viking name generator?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Our generator uses authentic Old Norse name elements and historically documented epithets from Viking Age sources (8th-11th centuries). While we combine elements that may not have been used together historically, each individual component is based on real Norse naming conventions and meanings."
      }
    },
    {
      "@type": "Question",
      "name": "Can I use generated Viking names for my book or game?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Yes, all names generated by this tool are free to use for any purpose including creative writing, games, role-playing, or personal projects. The names are based on historical public domain sources and common Norse naming patterns."
      }
    }
  ]
}
```

### Next.js Implementation

```typescript
export default function Page() {
  const faqSchema = {
    "@context": "https://schema.org",
    "@type": "FAQPage",
    "mainEntity": [
      {
        "@type": "Question",
        "name": "Question text here",
        "acceptedAnswer": {
          "@type": "Answer",
          "text": "Answer text here"
        }
      }
      // Add more Q&A pairs
    ]
  };

  return (
    <>
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(faqSchema) }}
      />
      {/* FAQ section HTML */}
    </>
  );
}
```

## HowTo Schema (Optional)

Use if you have step-by-step instructions on using the tool.

```json
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "How to Generate a Viking Name",
  "description": "Step-by-step guide to creating an authentic Viking name using our generator",
  "totalTime": "PT1M",
  "step": [
    {
      "@type": "HowToStep",
      "name": "Select Gender",
      "text": "Choose whether you want a male or female Viking name",
      "position": 1
    },
    {
      "@type": "HowToStep",
      "name": "Choose Style",
      "text": "Select a name style: Warrior, Peaceful, or Royal",
      "position": 2
    },
    {
      "@type": "HowToStep",
      "name": "Generate Name",
      "text": "Click the 'Generate My Viking Name' button to create your unique name",
      "position": 3
    },
    {
      "@type": "HowToStep",
      "name": "View Details",
      "text": "Read the meaning, Old Norse transliteration, and historical context of your generated name",
      "position": 4
    }
  ]
}
```

## BreadcrumbList Schema (Multi-Page Sites)

Use if you have breadcrumb navigation.

```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Home",
      "item": "https://yourdomain.com/"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Tools",
      "item": "https://yourdomain.com/tools"
    },
    {
      "@type": "ListItem",
      "position": 3,
      "name": "Viking Name Generator",
      "item": "https://yourdomain.com/tools/viking-name-generator"
    }
  ]
}
```

## Organization Schema (Site-Wide)

Put in header/footer. Defines your organization.

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Viking Name Forge",
  "url": "https://vikingnameforge.com",
  "logo": "https://vikingnameforge.com/logo.png",
  "sameAs": [
    "https://twitter.com/vikingnameforge",
    "https://facebook.com/vikingnameforge"
  ],
  "contactPoint": {
    "@type": "ContactPoint",
    "contactType": "Customer Support",
    "email": "support@vikingnameforge.com"
  }
}
```

## WebSite Schema (Site-Wide)

Enables sitelinks search box in Google results.

```json
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "Viking Name Forge",
  "url": "https://vikingnameforge.com",
  "potentialAction": {
    "@type": "SearchAction",
    "target": "https://vikingnameforge.com/search?q={search_term_string}",
    "query-input": "required name=search_term_string"
  }
}
```

## Multiple Schemas on One Page

You can combine multiple schemas using an array:

### Next.js

```typescript
export default function Page() {
  const schemas = [
    {
      "@context": "https://schema.org",
      "@type": "SoftwareApplication",
      // ... SoftwareApplication data
    },
    {
      "@context": "https://schema.org",
      "@type": "FAQPage",
      // ... FAQPage data
    }
  ];

  return (
    <>
      {schemas.map((schema, index) => (
        <script
          key={index}
          type="application/ld+json"
          dangerouslySetInnerHTML={{ __html: JSON.stringify(schema) }}
        />
      ))}
      {/* Your content */}
    </>
  );
}
```

### HTML

```html
<!-- Schema 1 -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  ...
}
</script>

<!-- Schema 2 -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  ...
}
</script>
```

## Validation

Always validate your schema markup:

**Google Rich Results Test:**
https://search.google.com/test/rich-results

**Schema.org Validator:**
https://validator.schema.org/

**Common Errors:**
- Missing required fields (name, description)
- Invalid date formats
- Incorrect @type values
- Malformed JSON (missing commas, quotes)

---

## Priority for Tool Sites

**Must Have:**
1. SoftwareApplication schema
2. FAQPage schema (if you have FAQ)

**Nice to Have:**
3. HowTo schema (if step-by-step guide)
4. Organization schema (site-wide)
5. BreadcrumbList (if multi-page)

**Skip:**
- Product schema (you're not selling physical products)
- Review schema (unless you have actual user reviews)
- Event schema (not applicable to tools)
